## Prerequisites

- [Node](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)
- [Expo](https://docs.expo.io/)

## Installing

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/custom-telegram.git

  # Enter folder
  $ cd custom-telegram

  # Install dependences
  $ yarn

  #Execute in Android
  $ yarn android

  #Execute in iOS
  $ yarn ios

```
